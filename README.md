# Scoop IAC repo

This is Scoop bucket for installing Windows tools.

## Bucket

To add tools from this bucket, first add the bucket:

```shell
scoop bucket add iac https://github.com/JamesWoolfenden/scoop.git
```

then you can install a tool:

```
scoop install pike
```

Supported tools:
- Pike 
- Sato
- Yor
